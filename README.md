# ars-github-actions-and-trivy-scanning
This is to demonstrate how to use GitHub Actions and Trivy to scan repos for vulnerabilities and exposed secrets.

Shows the use of GitHub Actions and Trivy to:

•	Scan source code for vulnerabilities and exposed secrets

•	Scan built image containers for vulnerabilities

•	Generate SBOMs 

•	Upload SARIF to GitHub Code Scanning (to the Security tab)

•	Upload all reports as build artifacts
